# Cassava Leaf Disease Classifier

AI component of an IoT system for cassava disease management (Raspberry Pi 3B
deployment). Upload a leaf photo to get a disease prediction with a calibrated
confidence score. EfficientNetB0 → TFLite, 5 classes.
